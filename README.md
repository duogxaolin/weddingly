# Weddingly

Nền tảng website thiệp cưới online **mã nguồn mở**, giúp các cặp đôi dễ dàng tạo và chia sẻ thiệp cưới của riêng mình.

> 💡 Dự án đang trong giai đoạn phát triển. Mọi đóng góp, góp ý đều được hoan nghênh!

## ✨ Tính năng

- 🎨 Nhiều mẫu giao diện đẹp, lãng mạn, dễ lựa chọn
- 📝 Tùy chỉnh nhanh thông tin cô dâu – chú rể, ngày giờ, địa điểm
- 🖼️ Hiển thị album ảnh cưới
- ✅ Form xác nhận tham dự (RSVP) cho khách mờ
- 📱 Tương thích thiết bị di động (responsive)
- ⚡ Dễ triển khai trên mọi hosting hỗ trợ PHP

## 🛠️ Công nghệ

- **Backend:** PHP
- **Frontend:** HTML5, CSS3, JavaScript
- **Giao diện:** Các template thiệp cưới tùy chỉnh

## 🚀 Cài đặt & chạy

1. Clone repo về máy:

   ```bash
   git clone https://github.com/duogxaolin/weddingly.git
   cd weddingly
   ```

2. Copy toàn bộ file lên hosting có hỗ trợ PHP (ví dụ: Apache/Nginx).

3. Truy cập đường dẫn tương ứng trên trình duyệt để xem website.
