# Weddingly

Nền tảng website thiệp cưới online mã nguồn mở, giúp các cặp đôi dễ dàng tạo và chia sẻ thiệp cưới của mình.

## Tính năng

- Nhiều mẫu giao diện lãng mạn, dễ tùy chỉnh
- Thay đổi thông tin cô dâu, chú rể, ngày giờ, địa điểm
- Album ảnh cưới
- Form xác nhận tham dự (RSVP)
- Tương thích di động (responsive)
- Chạy được trên mọi hosting hỗ trợ PHP

## Công nghệ

- PHP
- HTML5, CSS3, JavaScript
- Template thiệp cưới tùy chỉnh

## Cài đặt

1. Clone repository:

   ```bash
   git clone https://github.com/duogxaolin/weddingly.git
   cd weddingly
   ```

2. Copy toàn bộ file lên hosting có hỗ trợ PHP (Apache/Nginx).

3. Truy cập đường dẫn tương ứng trên trình duyệt.
